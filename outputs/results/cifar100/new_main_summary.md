### 因果论断的调整(两点偏离原预期)

1. **原预期 "同构 svhn 集成 AUROC 0.83-0.86,异构 ~0.94"**:实际异构只到 **0.8413**,差距从设想的 +0.10 缩到 **+0.034**。"AUROC 提升幅度"做主轴会显得太弱。
2. **原预期 "同构 OOD/ID 比值显著较小"**:实际两个集成的 OOD/ID 比值几乎相同(svhn 3.48 vs 3.27,cifar10 / textures 几乎一模一样)。**架构多样性放大的是 Δ 的绝对尺度而不是信噪比**。

### 更稳的因果说法

落在两个客观差异上:
- **Δ 绝对量级**整体 ~1.75 倍(异构 E_ID[Δ]=0.20,同构 0.11)
- **集成 ID acc** +1.98%(85.08% vs 83.10%)

这两条加起来支撑"架构多样性提供有益分歧"的叙事;**不要再写"信噪比被放大"**。

### ConvNeXt 重新定位

不是"伪影"也不是"全好",而是"**多数 OOD 提供正贡献、svhn 上中性偏负**":
- 异构集成在 cifar10 (0.8175 → 0.8442, +0.027) 和 textures (0.7916 → 0.8222, +0.031) 上都超过最佳单教师
- 只有在 svhn 上输给 densenet121 单教师 0.0097
- 这个细节适合写进 ablation,作为"集成不是无脑稳赚但 net effect 仍然正"的诚实交代

### 方法间对照(修复后)

- **`uniform` 与 `accuracy` 在四个核心指标上几乎完全一致**(ID acc 69.94 vs 69.37,cifar10 AUROC 0.7373 vs 0.7380,svhn 0.8376 vs 0.8215,textures 0.7672 vs 0.7604)。差距全部小于种子标准差。
- `learned_global` 在 ID acc 和 svhn AUROC 上系统性低 ~2% / ~10%,是**唯一显著的方法间差异**,但绝对量级仍小。
- **原预期"三种 MTKD 方法差异在种子方差量级以内,选最简单的 uniform 即可"在 `uniform` vs `accuracy` 上完全成立;`learned_global` 略偏弱但不足以推翻"选 uniform"这一结论**。

### "学生独立 OSR < 教师集成混合 OSR" 逐项核对

| OOD | 学生最佳 (uniform) | 混合 (集成 Energy) | 差距 |
|---|---:|---:|---:|
| cifar10 (近 OOD) | 0.7373 | 0.8442 | **+0.107** |
| svhn (远 OOD) | 0.8376 | 0.8413 | **+0.004** ← 几乎追平 |
| textures (远 OOD) | 0.7672 | 0.8222 | **+0.055** |
| **ID acc** | **69.94%** | **85.08%** | **+15.14%** |

### 意外发现

**svhn 上学生独立 OSR 几乎追上集成**(差距 0.004,在种子方差量级内),削弱了"混合方案在 OOD 检测上必然占优"的叙事。

### 混合方案必要性的论证调整

重心要从"OOD 检测被拉开差距"切到这两条:
1. **ID 分类质量** —— 学生 70% vs 集成 85%,差 15 个百分点,这是混合方案最强的卖点。
2. **近 OOD + 部分远 OOD** —— cifar10 +0.11、textures +0.05 都有显著差距,只在 svhn 这种纹理/数字差异极大的远 OOD 上失效。

## 跨实验 takeaway

1. **实验组 1 的 Jensen 间隙是新主线最稳的发现**:2.6 – 3.5x 的 OOD/ID 比值在 3 个 OOD 集上一致成立,数学+实证双锚点齐备,F1 直方图能撑起理论章节。
2. **实验组 2 的因果证据存在但比原预期弱一档**:Δ 绝对量级 +75-85%、集成 ID acc +2%;AUROC 提升量(+0.034 max)和 OOD/ID 比值这两个原预期主指标都不够强,理论章节的因果机制要从"信噪比放大"改写成"绝对尺度放大 + 集成 ID acc 增益"。
3. **ConvNeXt 在新框架下重新讲**:从"伪影"到"多数 OOD 正贡献、svhn 中性偏负"。
4. **混合推理方案的必要性**立足于 ID acc (+15%) 而非纯 OOD AUROC。
